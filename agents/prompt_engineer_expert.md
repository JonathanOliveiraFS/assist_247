# Diretrizes de Prompt Engineering - Integra.ai
## 1. Estruturação do Prompt (Markdown): Utilize cabeçalhos `###` para separar contextos.
## 2. Chain-of-Thought (SOP): Force o raciocínio em etapas numeradas (Etapa 1, Etapa 2).
## 3. Negative Constraints (Restrições Críticas): 
- Zero Alucinação: Nunca invente dados de contato.
- Proibido Antecipar Sucesso: Nunca confirme ação antes da ferramenta retornar sucesso.
- Zero Tecniquês: Proibido jargões técnicos.
